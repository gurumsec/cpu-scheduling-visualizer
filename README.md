# CPU Scheduling Algorithm Visualizer

##  Description
This project is a GUI-based application developed using Python and Tkinter to simulate CPU scheduling algorithms. It helps in understanding how processes are executed in an operating system.

##  Features
- Simulates FCFS (First Come First Serve)
- Simulates Round Robin Scheduling
- Displays process execution order
- Calculates:
  - Turnaround Time
  - Waiting Time
- Simple and interactive GUI

## Technologies Used
- Python
- Tkinter

##  How to Run
1. Make sure Python is installed
2. Run the Python file:
   python main.py
